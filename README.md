# Women Safety and Security System 🚨

The **Women Safety and Security System** is a web-based application designed to enhance safety and provide support during emergencies. This project integrates various modules for users, police, and administrators to ensure a responsive and effective system for handling complaints and emergencies.
URL:- https://raksha-kawatch.kesug.com/

---

## Features 🛡️

### User Module
- **Create Complaints**: Submit and manage incident reports.
- **SOS Button**: Instantly alert guardians and police with location details via SMS.
- **Guardian Management**: Add and manage trusted contacts for emergencies.
- **Complaint Status Tracking**: Monitor the status of submitted complaints.

### Police Module
- **Complaint Management**: View, update, and resolve complaints.
- **Incident Updates**: Provide real-time updates on incidents.

### Admin Module
- **Manage Users**: Oversee user registrations and activity.
- **Manage Police Accounts**: Register and manage police personnel.
- **View All Complaints**: Monitor and analyze user-reported issues.

---

## Technologies Used 💻

- **Frontend**: HTML, CSS, Bootstrap
- **Backend**: PHP
- **Database**: MySQL
- **API**: LocationIQ API for real-time location tracking
