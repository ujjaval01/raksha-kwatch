# Women Safety and Security System 🚨
This project is a comprehensive web application designed to enhance women's safety and security by providing quick and effective tools for emergency communication and reporting incidents. Built using PHP, MySQL, HTML, CSS, and Bootstrap, the system features three main modules: Admin, User, and Police, each with unique functionalities to ensure a robust and user-friendly experience.

🔑 Features
User Module
Create Complaints: Users can submit complaints directly to the system.
SOS Button: One-tap emergency alerts with location sharing via SMS to guardians and police.
Guardian Management: Add, view, and manage guardian contacts for emergency alerts.
Complaint Status: Track the progress and resolution of submitted complaints.
Police Module
Complaint Management: View and update the status of complaints.
Incident Updates: Provide real-time updates for incidents reported by users.
Admin Module
Manage Users: View and manage user accounts.
Manage Police: Register and manage police personnel.
View Complaints: Monitor all complaints submitted through the system.

👨‍💻 How to Run Locally
Install XAMPP to run PHP and MySQL locally.
Clone this repository:
bash
Copy code
git clone https://github.com/ujjaval01/women-safety-system.git
Place the project in the XAMPP htdocs folder.
Start Apache and MySQL in the XAMPP Control Panel.
Import the database.sql file into phpMyAdmin.
Access the application at http://localhost/women-safety-system.
