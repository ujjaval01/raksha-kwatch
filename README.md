    <h1>Women Safety and Security System 🚨</h1>
    <p>
        This project is a comprehensive web application designed to enhance women's safety and security by providing quick and effective tools for emergency communication and reporting incidents. Built using <span class="highlight">PHP</span>, <span class="highlight">MySQL</span>, <span class="highlight">HTML</span>, <span class="highlight">CSS</span>, and <span class="highlight">Bootstrap</span>, the system features three main modules: <strong>Admin</strong>, <strong>User</strong>, and <strong>Police</strong>, each with unique functionalities to ensure a robust and user-friendly experience.
    </p>

    <h2>🔑 Features</h2>
    <h3>User Module</h3>
    <ul>
        <li>Create Complaints: Submit complaints directly to the system.</li>
        <li>SOS Button: One-tap emergency alerts with location sharing via SMS to guardians and police.</li>
        <li>Guardian Management: Add, view, and manage guardian contacts for emergency alerts.</li>
        <li>Complaint Status: Track the progress and resolution of submitted complaints.</li>
    </ul>

    <h3>Police Module</h3>
    <ul>
        <li>Complaint Management: View and update the status of complaints.</li>
        <li>Incident Updates: Provide real-time updates for incidents reported by users.</li>
    </ul>

    <h3>Admin Module</h3>
    <ul>
        <li>Manage Users: View and manage user accounts.</li>
        <li>Manage Police: Register and manage police personnel.</li>
        <li>View Complaints: Monitor all complaints submitted through the system.</li>
    </ul>
